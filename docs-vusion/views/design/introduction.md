# 介绍

网易云平台就有较多的系统平台，不仅每个平台的量级较大且功能复杂，而且每个平台的迭代速度也很快，这就意味着设计和开发能在短时间内给出相应的解决方案。通过研究发现，各平台间存在很多相同的页面形态和组件，这样就可以通过归纳整理出一些可高复用性的内容。
